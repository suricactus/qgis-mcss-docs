Enumerators
LayerCrs

Optional layer CRS.
ProjectCrs

Current project CRS (if OTF reprojection enabled)
CurrentCrs

Current user selected CRS.
DefaultCrs

Global default QGIS CRS.
RecentCrs

Recently used CRS.
CrsNotSet

Not set (hidden by default)
