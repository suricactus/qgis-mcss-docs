QgsSimpleMarkerSymbolLayerBase class

Abstract base class for simple marker symbol layers.

Contents

  • Reference
    • Base classes
    • Derived classes

Handles creation of the symbol shapes but leaves the actual drawing of the symbols to subclasses.

Since

QGIS 2.16
