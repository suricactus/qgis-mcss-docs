QgsShadowEffect class

Base class for paint effects which offset, blurred shadows.

Contents

  • Reference
    • Base classes

Since

QGIS 2.9

Base classes

class QgsPaintEffect
Base class for visual effects which can be applied to QPicture drawings.

Derived classes
